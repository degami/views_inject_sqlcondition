Module
-------
Views Inject SQL Condition

CREDITS
--------

Forked by "Views Custom Conditions" module (https://drupal.org/project/views_custom_conditions)

DESCRIPTION
-------------

  This module allows inserting SQL statements in view query


BENEFITS
---------

 # Query with SQL statements

                                                            	 
INSTALLATION
-------------

  # Install the module
  # It will generate a field in views settings “ Index hint”
  # Insert your SQL statement in this field
  # Then save the views
  # This will inject the SQL statement


DEPENDENCY
----------
   "views_ui" module
   Download URL : http://drupal.org/project/views
